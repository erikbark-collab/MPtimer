Add your custom audio files here:

- beep.mp3
- go.mp3
- applause.mp3
- start-voice.mp3
- cheer-1.mp3
- cheer-2.mp3
- cheer-3.mp3
- Come on three sets left.m4a
- Warm up cheer.m4a
- Last set.m4a
- Random cheer.m4a
- Random cheer B.m4a
- Good job.m4a

If `beep.mp3` is missing, the app falls back to `go.mp3`, then to a generated beep.
If `go.mp3` is missing, the app falls back to a generated bright completion tone.
If any of the spoken cue files are missing, the app temporarily falls back to `go.mp3` as a placeholder.
Voice and cheer files fail silently when missing so the timer still works.

Add your custom audio files here:

- warning-beep.mp3
- go.wav
- start-voice.mp3
- cheer-1.mp3
- cheer-2.mp3
- cheer-3.mp3
- Come on three sets left.m4a
- Warm up cheer.m4a
- Last set.m4a
- Random cheer.m4a
- Good job.m4a

If `warning-beep.mp3` is missing, the app falls back to a generated beep.
If `go.wav` is missing, the app falls back to a generated bright completion tone.
If any of the spoken cue files are missing, the app temporarily falls back to `go.wav` as a placeholder.
Voice and cheer files fail silently when missing so the timer still works.
